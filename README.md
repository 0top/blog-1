## 博客 

[www.zerotop.top](https://www.zerotop.top "地址")


### 项目简介

这是我们团队zerotop的公用博客  
目前团队一共三个人，分别负责UI设计、前端和后端  
博客将会设计博客版块、设计版块、时间轴版块  



### 项目架构

项目采用前后端分离的开发方式

前端使用vue.js

后端使用spring

接口测试使用swagger-ui  

前后端代码将开源在github

### 使用方式

前端项目可以直接通过  
`
$ npm run dev
`    
打包通过  
`
$ npm run build  
`

后端可直接打包为war包，放到服务器上
